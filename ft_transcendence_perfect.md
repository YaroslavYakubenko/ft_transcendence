# 🧠 AI Instructions

## Context
AI can assist with:
- Code generation
- Documentation
- Explanations

- Outputs may be incorrect
- Context may be missing
- Answers may be generic

## Rules

- Never blindly copy code
- Be able to explain everything
- Use peers as validation layer

## Good Practice
```text
Ask → Test → Discuss → Improve
```

## Bad Practice
```text
Ask → Copy → Submit → Fail
```
